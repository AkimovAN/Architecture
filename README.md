# Model-View-Presenter sample for Android with Kotlin

This application used to explain how to use pattern [Model-View-Presenter](https://en.wikipedia.org/wiki/Model%E2%80%93view%E2%80%93presenter) (MVP) in a simple way.

# Useful links related to the MVP pattern
- [GUI Architectures - Martin Fowler](https://martinfowler.com/eaaDev/uiArchs.html)
- [MVP: Model-View-Presenter, The Taligent Programming Model for C++ and Java - Mike Potel](http://www.wildcrest.com/Potel/Portfolio/mvp.pdf)
- [Interactive Application Architecture Patterns](http://aspiringcraftsman.com/2007/08/25/interactive-application-architecture/)
- [Architecture samples by Google](https://github.com/googlesamples/android-architecture)
- [MVP introduction into Mosby - Hannes Dorfmann](http://hannesdorfmann.com/mosby/mvp/)
